# Ejercicios de Árboles Filogenéticos

Este repositorio contiene un notebook de Jupyter con ejercicios prácticos para el análisis y la manipulación de árboles filogenéticos.

---

## Contenido del Notebook (`exercises.ipynb`)

El notebook está diseñado para ayudar a comprender la estructura de los árboles filogenéticos, incluyendo sus nodos, ramas y clados, utilizando datos de la rama **Fungi**.

### Resumen del Ejercicio 1

El primer ejercicio se enfoca en las siguientes tareas:

* **Lectura de árboles**: Leer un árbol filogenético en formato Newick.
* **Análisis de estructura**: Obtener información detallada sobre los nodos, ramas y clados del árbol, como el número de nodos y terminales, y la longitud de las ramas.
* **Manipulación y Visualización**: Modificar atributos del árbol, como el nombre de una especie terminal (e.g., *Rozella\_allomycis*), ajustar la longitud de una rama (e.g., *Zoopagales* a **0.5**), y colorear ramas específicas (e.g., *Rhizophydiales* en rojo) para una visualización más informativa.
---

## Prerrequisitos

Para ejecutar este notebook, necesitarás tener instalado:

* **Python 3**
* **Jupyter Notebook** o **JupyterLab**
* **Biopython** (específicamente, el módulo `Phylo` para el manejo de árboles)
